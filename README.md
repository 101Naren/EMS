# Employee Managment System (EMS)

This is a fullstack web application which uses CRUD (Create, Read, Update, and Delete) to manage, organize, and streamline all employee-related data and daily workplace operations.

## Features:
  1) **Employee Management:** <br>
    a) Add, update, view, and manage employee records<br>
    b) Role-based access for administrators and employees<br>

  2) **Attendance Tracking:** <br>
    a) Monitor employee attendance<br>
    b) Attendance history and reporting<br>

  3) **Authentication & Authorization:** <br>
    a) Secure login system<br>
    b) Password hashing using bcrypt<br>
    c) Protected API routes using authentication middleware<br>

  4) **Dashboard Analytics:** <br>
    a) Admin Dashboard<br>
    b) Employee Dashboard<br>
    c) Attendance monitoring and statistics<br>

  5) **Responsive User Interface:** <br>
    a) Modern design built with Tailwind CSS<br>
    b) Optimized for desktop<br>

  6) **Real-Time Updates (if enabled):** <br>
    a) WebSocket-based communication for instant updates<br>

## Tech Stack:
  1) **Frontend:** <br>
    a) React<br>
    b) TypeScript<br>
    c) Vite<br>
    d) Tailwind CSS<br>
  2) **Backend:** <br>
    a) Node.js<br>
    b) Express.js<br>
    c) JavaScript<br>
  3) **Authentication & Security:** <br>
    a) bcrypt<br>
    b) Custom Authentication Middleware<br>
    c) Protected API Routes<br>
  4) **Database:** <br>
    a) SQL Database (MySQL/PostgreSQL)<br>
    b) Database Configuration through server/config/db.js<br>
  5) **Additional Tools:** <br>
    a) CORS<br>
    b) Body Parser<br>
    c) WebSockets (Socket.IO)<br>
    d) ESBuild<br>
    e) Rolldown/Vite Build Ecosystem<br>
